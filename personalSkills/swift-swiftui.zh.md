# Swift / SwiftUI

## 经验等级
高级 — 8 年以上 Swift，4 年以上 SwiftUI

## 项目
- **IHG iOS App 改造（2021–至今）**：使用 SwiftUI 对 IHG 酒店预订应用进行改造，服务全球数百万用户
- **RollOrNot（2025）**：完全使用 SwiftUI + SwiftData 构建的 AI 驱动学习伴侣
- **Lenscopy App**：使用 Objective-C 配合 Swift 集成实现摄像头处理流水线

## 核心优势
- Swift 6 并发（async/await、Actor、结构化并发）
- SwiftUI 架构（MVVM、组合、环境）
- 性能优化与崩溃率降低
- 使用 Swift Package Manager 进行模块化架构设计
- Combine 框架与响应式编程
