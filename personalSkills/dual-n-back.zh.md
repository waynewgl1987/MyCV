# 📖 回溯记忆训练 — 介绍

## 什么是回溯记忆训练？

回溯记忆训练是基于 Dual N-Back 认知训练理论开发的益智游戏。Dual N-Back 是认知科学中最广为人知的工作记忆训练方法，已被数百篇同行评审的科学论文证实对流体智力和工作记忆有显著的积极影响。

## 怎么玩？

每一轮你会在 3×3 格子上看到一个高亮的方块（位置）并听到一个字母发音（声音）。你需要记住这些刺激，然后判断当前轮次和第 N 轮之前的刺激是否相同：

- **位置相同** → 按「位置重复」
- **声音相同** → 按「声音重复」
- **都不同** → 不按按钮，等下一轮

## 通关标准

- 正确率 ≥ 90% → 获得满额积分
- 正确率 ≥ 80% → 获得一半积分
- 正确率 < 80% → 请继续练习

正确率 = (正确识别 + 正确拒判) ÷ 总判断数 × 100%

## 训练的好处

- 提升工作记忆容量
- 增强流体智力
- 对儿童智力发展有积极作用
- 帮助老年人延缓认知衰退
- 提升专注力和注意力控制
- 在日常生活中更好地处理复杂任务

## 科学依据

2008 年 Susanne Jaeggi 等人发表在《美国国家科学院院刊》(PNAS) 的研究首次证明 Dual N-Back 训练可以显著提升流体智力。此后大量研究验证了这一结论，尤其是对于儿童和青少年的认知发展效果显著。每天只需 15–20 分钟训练，持续 20 天即可见效。
