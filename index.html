Cross-platform remote desktop & AI bridge system built on GStreamer's multimedia pipeline, solving two key scenarios:

▸ AI Workflow Bridge
  • One command to control the remote host — execute tasks via OpenCode agent (write code, organize files), transfer organized files from remote to phone, or send files/images from phone to remote.
  • When OpenCode finishes a task, results are instantly pushed to your phone as a notification.
Using LiveKit as the real-time signaling & data channel layer for reliable message delivery between phone and remote host. All connections can optionally leverage NAT traversal (P2P hole punching) via free tools like Tailscale for end-to-end encryption — personal data stays fully isolated on your own devices.

▸ Remote Desktop Streaming
Real-time remote desktop viewing and control via GStreamer + WebRTC stack — an auxiliary feature to see and interact with the remote host screen. LAN latency 40–80ms @ 60fps with optimized multi-touch gesture controls for smooth operation.

Native clients for iOS/Android/macOS — one-click deployment.
基于 GStreamer 多媒体管道的跨平台远程桌面与 AI 桥接系统，主要解决两大场景：

▸ AI 工作流桥接
  • 一个指令即可控制远端主机 — 通过 OpenCode 执行任务、比如写代码，整理文件，整理后可将远端文件传输并保存到本地手机、手机也可发送文件或图片到远端等等。
  • 远端 OpenCode 完成任务后，结果会实时推送至手机通知。
基于 LiveKit 实现手机与远端主机之间的实时信令与数据通道，确保消息可靠投递。 所有连接可以选择基于外网穿透（P2P 打洞）技术，推荐免费穿透软件 Tailscale，实现端到端加密 — 个人数据完全隔离在自有设备中。

▸ 远程桌面串流
作为辅助功能，基于 GStreamer + WebRTC 技术栈实时查看并控制远端主机屏幕。局域网延迟 40–80ms @ 60fps，优化多点触控手势控制，操作流畅自然。

提供 iOS/Android/macOS 原生客户端，一键部署。
